# マニュアル用スクリーンショット置き場

`MANUAL.md` から参照される画像をここに置きます。実機（iPad 横向き）で撮るのが最も実態に合います。
以下のファイル名で保存すると、マニュアルにそのまま表示されます。

| ファイル名 | 撮る画面 |
|---|---|
| `01-start.png` | 起動直後（空状態。手順①が点灯、ビューファインダの角枠） |
| `02-load.png`  | 動画読込直後のプレビュー再生中、または先頭フレーム表示 |
| `03-scale.png` | スケール設定中（十字を基準の端に合わせた状態） |
| `04-origin.png`| 原点設定後（原点マーカーが映像上に表示） |
| `05-track.png` | トラッキング中（中央十字＋打った点の軌跡＋データ表に行が並ぶ） |
| `06-graph.png` | グラフ（速度 v–t などを表示、点が連動ハイライト） |
| `07-export.png`| データ出力ダイアログ（TSVコピー / TSV保存 / xlsx保存） |

## 撮り方のメモ
- iPad は横向き、ブラウザのアドレスバーを隠すと広く撮れます。
- スクショは iPad の「電源＋音量上」、PCブラウザなら開発者ツールのフルページキャプチャでも可。

> 注: 本リポジトリ作成時の自動環境ではブラウザ描画ビューポートが固定（DPR 2.2 / 702px）で、
> かつ自動再生のデコードが制限されていたため、実機相当のクリーンな自動キャプチャを取得できませんでした。
> そのため画像スロットのみ用意してあります。
